SPA Controls

My platform controls, a dataset, a grid and other form based controls

spaDataset

A JSON dataset object.

TIP: If using datasets for Lookups and Selects use the onload method to chain loads to ensure data is loaded before the main dataset is loaded. 

                        
                            <script>
                                dsBrowse = new spaDataset({
                                    'id': 'dsBrowse',
                                    'pk': 'account_id',
                                    'webservice': '../services/account_Services.asmx/GetAll',
                                    'rows': 30
                                });
                            </script>

Options

Options are passed via JavaScript.

Property Type Default Description
id string null Name given to the dataset
table string null Database table, required to maintain updates and post back to the server
pk string null Database Column that represents the primary key, required to maintain updates and post back to the server
webservice string null .asmx webservice that will return the result set
params json string null A fixed set of parameters used to retrieve data
filter json object filter[] Dynamic filters to restrict the retrieve of data based of serach criteria
rows int -1 Number of rows to use when paging -1 = no paging

Events

Event Parameters Description
onload null Fires when the dataset is returned

Methods

Method Parameters Description
Page (pageNo) Get the specified page of data
NextPage () Get the next page of data
PreviousPage () Get the previous page of data
RowNo () Returns the index of the current row
RowCount () Returns the number of rows in the dataset
GetRow (rowNo) Sets the current row to rowNo and returns the row as a JSON object
GetCurrentRow () Returns the current row as a JSON object
MoveFirst () Sets current row to first index (0)
MoveNext () Sets current row to current row + 1
MoveTo (rowNo) Sets current row to the specified rowNo
Reload (filters) reload the current page of data - using the given filter
AddRow () Add a new row to the dataset - not committed until Save()
EditColumn (column, value) Edit the column in the current row - not committed until Save()
DeleteRow () Delete the current row - not committed until Save()
Save () Commit any changes to the server

spaGrid

A Paged Grid control with search page 

                    
                        <div id="myGrid" spaGrid dataset="dsBrowse"></div>
                    
                
                        
                            <script>
                                $('#myGrid').spaGrid({
                                    'title': "Account Browse",
                                    'columns': myColumns,
                                    'filters': myFilters,
                                    'rowID': 'account_id',
                                    'height': 430,
                                    'newBtn': 'Add New Account',
                                    'newLocation': 'detail.htm'
                                });
                            </script>

Options

Options are passed via JavaScript.

Property Type Default Description
title string null Title displayed in the header of the grid
columns array [] A list of columns describing the layout of the grid
filters array [] Describes the serach panel and fields that can be searched
rowID string null The column that represents the id if the detail is drilled into i.e. the primary key of the data
height int null Height in px's of the grid
newBtn Boolean false Determines whether to put an Add New button on the grid
newLocation string null The page to navigate to if the add new button is clicked

Events

Event Parameters Description
onload null Fires when the grid is ready

spaColumns

Defines the columns on a spaGrid 

                    
                        <div id="myGrid" spaGrid dataset="dsBrowse"></div>
                    
                
                        
                            <script>
                                var myColumns = new Array(['Account Name', '50%', 'account_name', '', '', 'key'],
                                                          ['Account No', '50%', 'account_no', '', '', '']);
                            </script>

Options

Options are passed via JavaScript.

Property Type Default Description
Header string null Column Header
Width % null Column Width expressed as a percentage
Column string null Column name withing the dataset
dataType string null numeric - right align, datetime - date and time, date - date only or remove - Adds a red X button for deletes
dataFormat string null numeric:money
class string null Add a class to a column to trap on click events

spaFilters

Defines the filter columns to be used on the search page of a spaGrid 

                    
                        <div id="myGrid" spaGrid dataset="dsBrowse"></div>
                    
                
                        
                            <script>
                                var myFilters = new Array(['myLastName', 'account_name', 'Account Name', 'contains'],
                                                          ['myAccNo', 'account_no', 'Account No', 'contains']);
                            </script>

Options

Options are passed via JavaScript.

Property Type Default Description
Name string null Name used as the HTML element id
Column string null Database Column name to be searched
Label string null Label shown against the column on the Search panel
Condition string null How to apply the filter (contains, startswith, endswith, or =,>,< or <>)
